```python
import random
clouds = ['cumulus', 'cirrus', 'stratus', 'nimbus']
notes = {'cumulus': 'C', 'cirrus': 'E', 'stratus': 'A', 'nimbus': 'F#'}
for _ in range(8):
    cloud = random.choice(clouds)
    print(f"{cloud:8} → {notes[cloud]}")
```
Simulates scanning the sky—each cloud type becomes a musical note. Sky as sheet music.
